# Personal Website

Personal website of Yuqing (Jacky) Jiang.

Currently a lightweight static placeholder (plain HTML/CSS), responsive on both
desktop and mobile. Content and final structure are still being decided; the
site may later be migrated to a static site generator such as
[Hexo](https://github.com/hexojs/hexo).

## Local preview

Open `index.html` in a browser, or serve it:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
